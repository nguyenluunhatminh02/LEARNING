# Tiếng Hàn Trong Kinh Doanh (비즈니스 한국어)

## 1. Kính Ngữ Kinh Doanh

### 1.1 Bảng Chuyển Đổi Thường → Kinh Doanh

| Thường | Kinh doanh | Nghĩa |
|---|---|---|
| 만나다 | 뵙다 | Gặp |
| 말하다 | 말씀드리다 | Nói (với người trên) |
| 알다 | 알고 계시다 | Biết |
| 묻다 | 여쭈다 | Hỏi |
| 주다 | 드리다 | Cho/Đưa |
| 먹다/마시다 | 드시다 / 잡수시다 | Ăn/Uống (người trên) |
| 보다 | 보시다 | Xem (người trên) |
| 있다 | 계시다 | Có/Ở (người trên) |
| 없다 | 안 계시다 | Không có (người trên) |

### 1.2 Cấu Trúc Lịch Sự Kinh Doanh

| Mục đích | Cấu trúc | Ví dụ |
|---|---|---|
| Yêu cầu | -아/어 주시겠습니까? | 확인해 주시겠습니까? |
| Đề xuất | -(으)시면 어떨까요? | 회의를 내일로 하시면 어떨까요? |
| Xin phép | -아/어도 되겠습니까? | 질문해도 되겠습니까? |
| Từ chối lịch sự | -기가 어려울 것 같습니다 | 참석하기가 어려울 것 같습니다 |
| Đồng ý | 알겠습니다 / 그렇게 하겠습니다 | — |

## 2. Điện Thoại (전화)

### 2.1 Nhận Điện Thoại
```
여보세요, [회사명] [부서명] [이름]입니다.
(Xin chào, đây là [tên] phòng [phòng ban] công ty [tên].)

네, 말씀하세요.
(Vâng, xin mời nói.)

잠시만 기다려 주십시오. 연결해 드리겠습니다.
(Xin đợi một chút. Tôi sẽ kết nối cho.)

죄송합니다. 지금 자리에 안 계십니다.
(Xin lỗi, hiện không có ở chỗ.)

메시지를 남겨 주시겠습니까?
(Bạn có muốn để lại lời nhắn không?)
```

### 2.2 Gọi Điện Thoại
```
여보세요, [회사명]의 [이름]이라고 합니다.
(Xin chào, tôi là [tên] từ công ty [tên].)

[이름] 과장님 좀 부탁드립니다.
(Xin cho tôi nói chuyện với quản lý [tên].)

[건에 대해] 전화 드렸습니다.
(Tôi gọi về vấn đề [...].)

확인 후 다시 연락 드리겠습니다.
(Sau khi xác nhận, tôi sẽ liên lạc lại.)
```

## 3. Email Kinh Doanh

### 3.1 Cấu Trúc Email Chuẩn

```
[받는 사람] 김과장님께

안녕하십니까, [회사명] [부서] [이름]입니다.

[본문 — Nội dung chính]

바쁘신 중에 죄송합니다만, 확인 부탁드립니다.
(Xin lỗi phiền khi bận, nhờ xác nhận.)

감사합니다.
[이름] 드림
[연락처]
```

### 3.2 Mẫu Email Thường Dùng

**Gửi tài liệu:**
```
제목: [프로젝트명] 관련 자료 송부의 건

김부장님께,

안녕하십니까, 마케팅팀 이영수입니다.
요청하신 자료를 첨부하여 보내 드립니다.
추가 사항이 있으시면 말씀해 주십시오.

감사합니다.
이영수 드림
```

**Xin lỗi:**
```
제목: 납기 지연에 대한 사과

박과장님께,

안녕하십니까, 생산팀 홍길동입니다.
금번 납기가 지연된 점 깊이 사과드립니다.
현재 최선을 다해 조치 중이며, [날짜]까지 완료하겠습니다.
다시 한번 죄송합니다.

홍길동 드림
```

**Lên lịch họp:**
```
제목: 회의 일정 안내

관계자 여러분께,

안녕하십니까, 기획팀 김민수입니다.
아래와 같이 회의를 진행하고자 합니다.

- 일시: 2024년 3월 15일 (금) 오후 2시
- 장소: 본사 3층 회의실
- 안건: 2분기 사업 계획 검토
- 참석 대상: 각 팀장

참석 가능 여부를 금주 내로 회신 부탁드립니다.

김민수 드림
```

## 4. Họp & Thuyết Trình

### 4.1 Mở Đầu Cuộc Họp
```
그럼 회의를 시작하겠습니다.
(Xin bắt đầu cuộc họp.)

오늘 안건은 [주제]에 대한 것입니다.
(Chủ đề hôm nay là về [...].)

먼저 [이름]님께서 발표해 주시겠습니까?
(Trước tiên xin mời [tên] trình bày.)
```

### 4.2 Trình Bày Ý Kiến
| Mục đích | Tiếng Hàn |
|---|---|
| Nêu ý kiến | 제 생각에는 ~인 것 같습니다 |
| Đồng ý | 저도 같은 생각입니다 |
| Phản đối lịch sự | 좋은 의견이지만, ~은/는 어떨까요? |
| Đề xuất | ~하는 것이 좋겠습니다 |
| Hỏi ý kiến | 이 점에 대해 어떻게 생각하십니까? |
| Xin thêm thông tin | 좀 더 자세히 설명해 주시겠습니까? |

### 4.3 Kết Thúc Cuộc Họp
```
오늘 논의한 내용을 정리하면 다음과 같습니다.
(Tóm tắt nội dung thảo luận hôm nay như sau.)

다음 회의는 [날짜]에 진행하겠습니다.
(Cuộc họp tiếp theo sẽ vào [ngày].)

수고하셨습니다.
(Mọi người đã vất vả.)
```

## 5. Thuyết Trình (발표)

### Cấu Trúc Thuyết Trình
```
[인사] 안녕하십니까, [소속] [이름]입니다.
[주제] 오늘 발표 주제는 "..."입니다.
[목차] 발표는 세 부분으로 구성되어 있습니다.

[본론 1] 먼저, ...에 대해 말씀드리겠습니다.
         이 그래프를 보시면...
         
[본론 2] 다음으로, ...에 대해 설명하겠습니다.
         구체적인 예를 들면...

[본론 3] 마지막으로, ...

[결론] 이상으로 발표를 마치겠습니다.
       질문이 있으시면 말씀해 주십시오.
       감사합니다.
```

## 6. Phỏng Vấn Xin Việc

### 6.1 Câu Hỏi Thường Gặp

| Câu hỏi | Mẫu trả lời |
|---|---|
| 자기소개를 해 주세요. | 안녕하십니까, [이름]입니다. [학교]에서 [전공]을 전공했습니다... |
| 지원 동기가 무엇입니까? | 귀사의 [특징]에 매력을 느꼈습니다... |
| 장점과 단점은 무엇입니까? | 장점은 [...]이며, 단점은 [...]이지만 개선하고 있습니다... |
| 10년 후 계획은? | [구체적 목표]를 달성하고 싶습니다... |
| 마지막으로 하고 싶은 말? | 기회를 주시다면 최선을 다하겠습니다 |

### 6.2 Mẫu Tự Giới Thiệu (자기소개)
```
안녕하십니까, [이름]이라고 합니다.
[대학교]에서 [전공]을 전공하고 올해 졸업했습니다.
대학 시절 [경험/인턴/프로젝트]을 통해 [역량]을 키웠습니다.
특히 [구체적 성과]를 달성한 경험이 있습니다.
이러한 경험을 바탕으로 귀사에서 [목표]에 기여하고 싶습니다.
감사합니다.
```

## 8. Bài Tập Thực Hành

### Bài 1: Viết email
Viết email báo cáo tiến độ dự án cho quản lý (과장님):
- Dự án hoàn thành 70%
- Gặp khó khăn về thời gian
- Đề xuất gia hạn 1 tuần

### Bài 2: Roleplay điện thoại
Bạn gọi đến công ty 삼성전자, muốn nói chuyện với 김과장님 về đơn hàng mới. Viết script cho cả 2 bên.

### Bài 3: Thuyết trình 3 phút
Chủ đề: "우리 회사의 신제품 소개" (Giới thiệu sản phẩm mới của công ty)
- Dùng cấu trúc thuyết trình chuẩn
- Bao gồm số liệu và so sánh
