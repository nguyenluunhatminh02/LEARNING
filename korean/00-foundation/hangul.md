## 3. Cấu Trúc Âm Tiết

Hangul ghép chữ theo khối vuông:

### Dạng 1: Phụ âm + Nguyên âm (CV)
```
ㅎ + ㅏ = 하 (ha)
ㄴ + ㅏ = 나 (na)
```

### Dạng 2: Phụ âm + Nguyên âm + Phụ âm cuối (CVC)
```
ㅎ + ㅏ + ㄴ = 한 (han)
ㄱ + ㅜ + ㄱ = 국 (guk)
```

### Dạng 3: Phụ âm + Nguyên âm + 2 Phụ âm cuối (CVCC)
```
ㄷ + ㅏ + ㄹ + ㄱ = 닭 (dak - gà)
```

### Vị Trí Ghép Chữ
```
Nguyên âm dọc (ㅏ,ㅓ,ㅣ...):     Nguyên âm ngang (ㅗ,ㅜ,ㅡ...):
┌───┬───┐                          ┌───────┐
│ C │ V │                          │   C   │
├───┴───┤                          ├───────┤
│  (C)  │                          │   V   │
└───────┘                          ├───────┤
                                   │  (C)  │
                                   └───────┘
```
