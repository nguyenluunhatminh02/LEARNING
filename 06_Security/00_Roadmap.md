# 🛡️ 06 — Security & Compliance Curriculum

## Mục tiêu tổng quan
Trang bị kiến thức bảo mật từ lập trình an toàn đến architecture security, giúp bạn bảo vệ hệ thống ở cấp độ CTO/Staff Engineer.

---

## Cấu trúc: 10 bài học — 3 phần

### Phần 1: Application Security (Bài 01–04)
| Bài | Chủ đề | Nội dung chính |
|-----|--------|----------------|
| 01 | OWASP Top 10 & Secure Coding | Injection, XSS, CSRF, SSRF, broken auth |
| 02 | Authentication & Authorization | OAuth2, OIDC, JWT, RBAC, ABAC, session management |
| 03 | Cryptography Fundamentals | Hashing, encryption, TLS, key management |
| 04 | API Security | Rate limiting, input validation, API keys, CORS |

### Phần 2: Infrastructure Security (Bài 05–07)
| Bài | Chủ đề | Nội dung chính |
|-----|--------|----------------|
| 05 | Network & Cloud Security | VPC, firewalls, WAF, DDoS protection, Zero Trust |
| 06 | Container & K8s Security | Image scanning, pod security, RBAC, secrets |
| 07 | Supply Chain Security | Dependency scanning, SBOM, signed builds |

### Phần 3: Governance & Leadership (Bài 08–10)
| Bài | Chủ đề | Nội dung chính |
|-----|--------|----------------|
| 08 | Security Architecture & Threat Modeling | STRIDE, attack trees, security design review |
| 09 | Compliance & Data Privacy | GDPR, SOC2, PCI-DSS, data classification |
| 10 | Security Program Leadership | Security culture, incident response, DevSecOps |

---

## Lộ trình học

```
Tháng 1-2: Application Security (Bài 01-04)
  → Hands-on: Fix vulnerabilities in intentionally vulnerable app (DVWA, Juice Shop)

Tháng 3: Infrastructure Security (Bài 05-07)
  → Hands-on: Harden K8s cluster, setup WAF

Tháng 4: Governance & Leadership (Bài 08-10)
  → Hands-on: Build threat model cho real system, create security program
```
