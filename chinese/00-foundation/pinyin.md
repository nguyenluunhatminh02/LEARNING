# Pinyin — Bảng Phiên Âm Cơ Bản

Pinyin là nền tảng để đọc, nghe và phát âm tiếng Trung. Mục tiêu: **nắm chắc trong 2 tuần đầu**.

## 1. Thanh Mẫu (Initials)

| Nhóm | Thanh mẫu |
|------|-----------|
| Cơ bản | b, p, m, f |
| Đầu lưỡi | d, t, n, l |
| Gốc lưỡi | g, k, h |
| Răng | j, q, x |
| Uốn lưỡi | zh, ch, sh, r |
| Răng trước | z, c, s |
| Bán nguyên âm | y, w |

## 2. Vận Mẫu (Finals)

| Nhóm | Vận mẫu tiêu biểu |
|------|-------------------|
| Đơn | a, o, e, i, u, ü |
| Kép | ai, ei, ao, ou |
| Mũi trước | an, en, in, un, ün |
| Mũi sau | ang, eng, ing, ong |
| Kết hợp | ia, ie, iao, iu, ua, uo, uai, ui, üe |

## 3. Các Cặp Dễ Nhầm

| Cặp | Ghi chú |
|-----|---------|
| j / q / x | Đi với `i`, `ü`, âm mảnh hơn `zh/ch/sh` |
| z / c / s | Đầu lưỡi gần răng, không uốn |
| zh / ch / sh | Uốn lưỡi rõ hơn |
| n / ng | `n` kết thúc trước, `ng` giữ hơi ở mũi lâu hơn |
| u / ü | `ü` phải chu môi, như âm giữa `i` và `u` |

## 4. Mẹo Học Pinyin

1. Học theo nhóm dễ nhầm, không học rời từng âm.
2. Nghe mẫu chuẩn rồi đọc nhại lại ngay.
3. Ghi âm chính mình để so với audio mẫu.
4. Với `zh/ch/sh/r`, ưu tiên khẩu hình đúng hơn tốc độ.
5. Với `j/q/x` và `ü`, luyện trước gương 5 phút mỗi ngày.

## 5. Lịch Trình 14 Ngày

| Ngày | Nội dung | Mục tiêu |
|------|----------|----------|
| 1-2 | b p m f + a o e i u ü | Nhận diện và đọc chậm |
| 3-4 | d t n l + ai ei ao ou | Đọc đúng cặp hơi bật |
| 5-6 | g k h + an en ang eng | Phân biệt mũi trước/sau |
| 7-8 | j q x + ia ie iao iu | Làm quen nhóm âm khó |
| 9-10 | zh ch sh r + ua uo uai ui | Uốn lưỡi đúng khẩu hình |
| 11-12 | z c s + in ing ong | Nghe phân biệt nhanh |
| 13 | Ôn toàn bộ | Đọc bảng pinyin liên tục |
| 14 | Tự kiểm tra | Đọc 30 âm không nhìn bảng |
