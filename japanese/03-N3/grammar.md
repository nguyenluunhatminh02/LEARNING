# Ngữ Pháp N3 — 180 Mẫu Câu Trung Cấp

## Nhóm 1: Biểu Đạt Quan Hệ & Phạm Vi

### 1. ～において / における — Tại / Trong
```
日本において、桜は特別な花です。
Tại Nhật Bản, hoa anh đào là loài hoa đặc biệt.

国際社会における日本の役割
Vai trò của Nhật Bản trong xã hội quốc tế
```

### 2. ～に対して — Đối Với
```
先生は学生に対して厳しいです。
Thầy nghiêm khắc đối với học sinh.

質問に対して、丁寧に答えました。
Đã trả lời lịch sự đối với câu hỏi.
```

### 3. ～について — Về
```
日本文化について研究しています。
Tôi đang nghiên cứu về văn hóa Nhật.
```

### 4. ～によって / による — Do / Bởi / Tùy Theo
```
この小説は村上春樹によって書かれた。
Tiểu thuyết này được viết bởi Murakami Haruki.

国によって、文化が違います。
Tùy theo quốc gia mà văn hóa khác nhau.
```

### 5. ～として — Với Tư Cách
```
留学生として日本に来ました。
Tôi đến Nhật với tư cách du học sinh.
```

### 6. ～に関して / に関する — Liên Quan Đến
```
この問題に関して、もう少し調べます。
Về vấn đề này, tôi sẽ tra cứu thêm.
```

### 7. ～をもとに — Dựa Trên
```
実話をもとに作られた映画です。
Phim được làm dựa trên chuyện thật.
```

### 8. ～を通じて / を通して — Thông Qua
```
ボランティアを通じて、多くのことを学んだ。
Thông qua tình nguyện, tôi đã học được nhiều.
```

### 9. ～にとって — Đối Với (Quan Điểm)
```
私にとって、家族が一番大切です。
Đối với tôi, gia đình quan trọng nhất.
```

### 10. ～にかけて — Suốt Từ...Đến
```
3月から4月にかけて、桜が咲きます。
Suốt từ tháng 3 đến tháng 4, hoa anh đào nở.
```

## Nhóm 2: Nguyên Nhân & Kết Quả

### 11. ～ため(に) — Vì / Để
```
台風のために、電車が止まりました。 (Nguyên nhân)
Vì bão, tàu ngừng chạy.

健康のために、毎日運動します。 (Mục đích)
Để khỏe mạnh, tôi tập thể dục mỗi ngày.
```

### 12. ～おかげで — Nhờ Có
```
先生のおかげで、合格しました。
Nhờ có thầy mà tôi đỗ.
```

### 13. ～せいで — Vì Lỗi / Do (Tiêu cực)
```
雨のせいで、試合が中止になった。
Do mưa mà trận đấu bị hủy.
```

### 14. ～わけだ — Hóa Ra / Tức Là
```
A: 彼は10年アメリカに住んでいたんです。  
B: だから英語がうまいわけだ。
Hóa ra vì thế mà anh ấy giỏi tiếng Anh.
```

### 15. ～ことから — Từ Việc
```
形が星に似ていることから、「星の砂」と名付けられた。
Từ việc hình dáng giống ngôi sao nên được đặt tên là "Cát sao".
```

### 16. ～以上 — Đã...Thì
```
約束した以上、守らなければなりません。
Đã hứa thì phải giữ.
```

### 17. ～上で — Sau Khi / Trong Khi
```
よく考えた上で、決めます。
Sau khi suy nghĩ kỹ, tôi sẽ quyết định.
```

## Nhóm 3: Điều Kiện & Giả Thiết

### 18. ～さえ...ば — Chỉ Cần...Thì
```
お金さえあれば、旅行できます。
Chỉ cần có tiền thì đi du lịch được.
```

### 19. ～としたら / とすれば — Nếu Giả Sử
```
もし100万円あるとしたら、何をしますか。
Nếu giả sử có 1 triệu yên, bạn sẽ làm gì?
```

### 20. ～ないことには — Nếu Không...Thì
```
実際に行かないことには、わからない。
Nếu không thực sự đi thì không hiểu.
```

### 21. ～次第 — Ngay Khi / Tùy Theo
```
届き次第、連絡します。 (Ngay khi)
Ngay khi nhận được, tôi sẽ liên lạc.

結果は努力次第です。 (Tùy theo)
Kết quả tùy theo nỗ lực.
```

### 22. ～限り — Trong Phạm Vi / Miễn Là
```
私が知っている限り、彼は独身です。
Theo những gì tôi biết, anh ấy độc thân.
```

## Nhóm 4: Phỏng Đoán & Xác Nhận

### 23. ～に違いない — Chắc Chắn Là
```
彼は犯人に違いない。
Anh ta chắc chắn là thủ phạm.
```

### 24. ～おそれがある — Có Nguy Cơ
```
台風が来るおそれがあります。
Có nguy cơ bão đến.
```

### 25. ～っぽい — Có Vẻ / Hay
```
彼女は怒りっぽい。— Cô ấy hay nóng tính.
今日は秋っぽい天気ですね。— Thời tiết hôm nay có vẻ thu.
```

### 26. ～気味 — Hơi / Có Phần
```
最近、風邪気味です。
Gần đây hơi bị cảm.
```

### 27. ～向き — Hướng Về / Phù Hợp
```
この本は初心者向きです。
Sách này phù hợp người mới.
```

### 28. ～向け — Dành Cho
```
子供向けの番組
Chương trình dành cho trẻ em.
```

## Nhóm 5: Mức Độ & Phạm Vi

### 29. ～てたまらない — Quá...Không Chịu Nổi
```
暑くてたまらない。— Nóng không chịu nổi.
会いたくてたまらない。— Nhớ quá không chịu nổi.
```

### 30. ～てしかたがない — Không Thể Chịu Được
```
心配でしかたがない。
Lo lắng không thể chịu được.
```

### 31. ～てならない — Cứ...Mãi
```
彼のことが気になってならない。
Cứ nghĩ mãi về anh ấy.
```

### 32. ～ばかりか — Không Chỉ...Mà Còn
```
漢字ばかりか、ひらがなも読めません。
Không chỉ kanji, ngay hiragana cũng không đọc được.
```

### 33. ～に限らず — Không Chỉ Giới Hạn
```
日本に限らず、多くの国で人口問題がある。
Không chỉ Nhật, nhiều nước cũng có vấn đề dân số.
```

### 34. ～はもちろん — Đương Nhiên
```
英語はもちろん、日本語も話せます。
Đương nhiên tiếng Anh, cả tiếng Nhật cũng nói được.
```

### 35. ～どころか — Nói Gì Đến
```
旅行どころか、仕事も行けない状態です。
Nói gì đến du lịch, đến đi làm cũng không được.
```

## Nhóm 6: Biểu Đạt Khác

### 36. ～ざるを得ない — Buộc Phải
```
雨なので、中止せざるを得ない。
Vì mưa nên buộc phải hủy.
```

### 37. ～わけにはいかない — Không Thể (Đạo Đức/Xã Hội)
```
友達の秘密を話すわけにはいかない。
Không thể nói bí mật của bạn.
```

### 38. ～ものだ — Nên / Đáng Lẽ
```
人の悪口を言うものではない。
Không nên nói xấu người khác.

子供の頃、よくこの川で遊んだものだ。(Hoài niệm)
Hồi nhỏ, hay chơi ở sông này.
```

### 39. ～つつある — Đang Dần
```
景気は回復しつつある。
Kinh tế đang dần hồi phục.
```

### 40. ～一方だ — Càng Ngày Càng
```
物価は上がる一方だ。
Giá cả càng ngày càng tăng.
```
