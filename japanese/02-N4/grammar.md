# Ngữ Pháp N4 — 130 Mẫu Câu Trung Cấp Cơ Bản

## Phần 1: Thể Điều Kiện (条件形)

### 1. ～たら — Nếu...thì
**Cấu trúc**: Thể た + ら
```
あした あめが ふったら、いきません。
Nếu ngày mai mưa thì tôi không đi.

やすかったら、かいます。
Nếu rẻ thì mua.
```

### 2. ～ば — Nếu (giả thiết)
**Cấu trúc**: Động từ (え段) + ば
```
いけば、わかります。(いく→いけば)
Nếu đi thì sẽ hiểu.

やすければ、かいます。(やすい→やすければ)
Nếu rẻ thì mua.
```

### 3. ～と — Nếu (tự nhiên/thói quen)
```
はるに なると、さくらが さきます。
Khi đến mùa xuân, hoa anh đào nở.

このボタンを おすと、ドアが あきます。
Nhấn nút này thì cửa mở.
```

### 4. ～なら — Nếu (nói về chủ đề)
```
にほんに いくなら、きょうとに いったほうがいい。
Nếu đi Nhật thì nên đi Kyoto.
```

## Phần 2: Thụ Động & Sai Khiến

### 5. 受身形 (Thụ Động) ～(ら)れる
| Nhóm | Cách chia | Ví dụ |
|------|-----------|-------|
| I | う→あれる | のむ→のまれる |
| II | る→られる | たべる→たべられる |
| III | する→される、くる→こられる | |

```
わたしは せんせいに ほめられました。
Tôi được thầy khen.

あめに ふられました。
Bị mưa (dính mưa).

でんしゃで あしを ふまれました。
Bị giẫm chân trên tàu.
```

### 6. 使役形 (Sai Khiến) ～(さ)せる
| Nhóm | Cách chia | Ví dụ |
|------|-----------|-------|
| I | う→あせる | のむ→のませる |
| II | る→させる | たべる→たべさせる |
| III | する→させる、くる→こさせる | |

```
せんせいは がくせいに ほんを よませました。
Thầy bắt học sinh đọc sách.

ははは こどもに やさいを たべさせます。
Mẹ bắt con ăn rau.
```

### 7. 使役受身形 (Sai khiến + Thụ động) ～(さ)せられる
```
わたしは ははに やさいを たべさせられました。
Tôi bị mẹ bắt ăn rau.
```

## Phần 3: Phỏng Đoán & Suy Luận

### 8. ～そうです (Dự đoán bề ngoài)
**Cấu trúc**: Vます / い Adj (bỏ い) / な Adj + そうです

```
あめが ふりそうです。— Trông sắp mưa.
このケーキは おいしそうです。— Bánh này trông ngon.
かのじょは げんきそうです。— Cô ấy trông khỏe.
```
**Ngoại lệ**: いい → よさそう、ない → なさそう

### 9. ～ようです / ～みたいです — Có Vẻ Như
```
かれは びょうきの ようです。— Anh ấy có vẻ bệnh.
かのじょは うれしい みたいです。— Cô ấy có vẻ vui.
```

### 10. ～らしいです — Nghe Nói / Đúng Kiểu
```
あした あめらしいです。— Nghe nói ngày mai mưa.
かれは にほんじんらしい にほんじんです。— Anh ấy là người Nhật đúng kiểu.
```

### 11. ～かもしれません — Có Thể
```
あした あめが ふるかもしれません。
Ngày mai có thể mưa.
```

### 12. ～はずです — Đáng Lẽ Phải
```
かれは もう ついた はずです。— Anh ấy đáng lẽ đã đến rồi.
そんな はずは ありません。— Không thể nào như vậy.
```

### 13. ～でしょう (Nâng cao)
```
あしたは はれる でしょう。— Ngày mai chắc nắng.
もう しっている でしょう？ — Bạn biết rồi phải không?
```

## Phần 4: Biểu Đạt Ý Định & Mục Đích

### 14. ～ことにする — Quyết Định
```
まいにち うんどうする ことに しました。
Tôi đã quyết định tập thể dục mỗi ngày.
```

### 15. ～ことになる — Được Quyết Định (Bị Động)
```
らいげつ にほんに いく ことに なりました。
Tháng sau tôi sẽ đi Nhật (do công ty/hoàn cảnh quyết định).
```

### 16. ～つもりです (Nâng cao)
```
だいがくに いく つもりは ありません。
Tôi không có ý định đi đại học.
```

### 17. ～ために — Để (Mục Đích)
```
にほんに りゅうがくする ために、おかねを ためています。
Để du học Nhật, tôi đang tiết kiệm.
```

### 18. ～ように — Để / Sao Cho
```
おくれないように、はやく おきます。
Để không trễ, tôi dậy sớm.
```

### 19. ～ようにする — Cố Gắng (Thói Quen)
```
まいにち やさいを たべるように しています。
Tôi cố gắng ăn rau mỗi ngày.
```

### 20. ～ようになる — Trở Nên Có Thể
```
にほんごが はなせるように なりました。
Tôi đã nói được tiếng Nhật.
```

## Phần 5: Trạng Thái & Thời Điểm

### 21. ～ているところです — Đang (đúng lúc)
```
いま ごはんを たべている ところです。
Tôi đang ăn cơm (đúng lúc).
```

### 22. ～るところです — Sắp
```
いま でかける ところです。— Tôi sắp ra ngoài.
```

### 23. ～たところです — Vừa Mới
```
いま かえった ところです。— Tôi vừa mới về.
```

### 24. ～たばかりです — Vừa Mới (Nhấn Mạnh)
```
にほんに きた ばかりです。
Tôi vừa mới đến Nhật.
```

### 25. ～てある — Đã Được Làm (Trạng Thái)
```
まどが あけて あります。— Cửa sổ đã được mở.
ビールが かって あります。— Bia đã được mua sẵn.
```

### 26. ～ておく — Làm Trước / Chuẩn Bị
```
ホテルを よやくして おきます。
Đặt khách sạn trước.
```

### 27. ～てしまう — Hoàn Thành / Tiếc Nuối
```
しゅくだいを わすれて しまいました。
Quên mất bài tập (tiếc quá).

ぜんぶ たべて しまいました。
Ăn hết rồi.
```

## Phần 6: Kết Nối & Biểu Đạt Phức Tạp

### 28. ～し — Liệt Kê Lý Do
```
あの レストランは やすいし、おいしいし、よく いきます。
Nhà hàng kia rẻ, lại ngon, nên hay đi.
```

### 29. ～のに — Mặc Dù (Bất Ngờ/Thất Vọng)
```
やくそく したのに、きませんでした。
Mặc dù đã hứa mà không đến.
```

### 30. ～ても — Dù Cho
```
あめが ふっても、いきます。
Dù mưa cũng đi.

いくら たべても、ふとりません。
Dù ăn bao nhiêu cũng không béo.
```

### 31. ～のに (Mục đích) — Để
```
この はさみは かみを きるのに つかいます。
Cái kéo này dùng để cắt giấy.
```

### 32. ～ないで — Không Làm Mà
```
あさごはんを たべないで、がっこうに いきました。
Không ăn sáng mà đi học.
```

### 33. ～ずに — Không Làm Mà (Văn viết)
```
じしょを つかわずに、よみました。
Không dùng từ điển mà đọc.
```

### 34. ～ているあいだに — Trong Khi
```
ははが りょうりを しているあいだに、そうじを しました。
Trong khi mẹ nấu ăn, tôi dọn nhà.
```

### 35. ～たとたんに — Vừa...Thì Ngay
```
いえを でた とたんに、あめが ふりだしました。
Vừa ra khỏi nhà thì mưa.
```

## Phần 7: Kính Ngữ Cơ Bản (敬語)

### 36. お + Vます + になる — Kính Ngữ (尊敬語)
```
せんせいは もう おかえりに なりました。
Thầy đã về rồi.
```

### 37. お + Vます + する — Khiêm Nhường (謙譲語)
```
わたしが おもちします。
Tôi xin mang giúp.
```

### 38. Kính Ngữ Đặc Biệt

| Bình thường | Kính ngữ (尊敬語) | Khiêm nhường (謙譲語) |
|------------|-------------------|---------------------|
| いる | いらっしゃる | おる |
| いく/くる | いらっしゃる | まいる |
| たべる/のむ | めしあがる | いただく |
| いう | おっしゃる | もうす |
| みる | ごらんになる | はいけんする |
| する | なさる | いたす |
| しる | ごぞんじ | ぞんじる |
| あげる | — | さしあげる |
| もらう | — | いただく |
| くれる | くださる | — |

## Phần 8: Các Mẫu Câu Bổ Sung (39-80)

### 39. ～ことがある — Có Lúc
```
ときどき にほんりょうりを つくる ことが あります。
Thỉnh thoảng tôi có nấu món Nhật.
```

### 40. ～ようにする — Chú Ý Làm
```
おそく ならないように しましょう。
Hãy chú ý đừng đến trễ.
```

### 41. ～てくる / ～ていく — Chuyển Biến
```
にほんご が だんだん わかってきました。— Dần dần hiểu tiếng Nhật.
これから さむく なっていきます。— Sẽ dần trở nên lạnh.
```

### 42. ～ことは ～が — Thì ... Nhưng
```
おいしいことは おいしいですが、たかいです。
Ngon thì ngon nhưng đắt.
```

### 43. ～わけではない — Không Phải Là
```
にほんが きらいな わけではありません。
Không phải là ghét Nhật Bản.
```

### 44. ～ということです — Có Nghĩa Là
```
つまり、むりだ ということですか。
Nghĩa là không được sao?
```

### 45. ～って — Nghe Nói (Khẩu Ngữ)
```
あした やすみだって。— Nghe nói ngày mai nghỉ.
```

### 46. ～てほしい — Muốn Ai Đó Làm
```
しずかにしてほしいです。— Tôi muốn bạn im lặng.
```

### 47. ～させてください — Cho Phép Tôi
```
わたしに やらせてください。— Hãy cho tôi làm.
```

### 48. ～(さ)せてもらう / いただく — Xin Phép Làm
```
おさきに かえらせていただきます。
Xin phép về trước.
```

### 49. ～てくれない？ — Giúp Được Không? (Thân Mật)
```
ちょっと てつだってくれない？
Giúp một chút được không?
```

### 50. ～ずにはいられない — Không Thể Không
```
おもしろくて わらわずには いられません。
Buồn cười quá không nhịn được.
```
