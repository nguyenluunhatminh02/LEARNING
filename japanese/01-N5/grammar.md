# Ngữ Pháp N5 — 80 Mẫu Câu Cơ Bản

## 1. Câu Khẳng Định với です
**Cấu trúc**: A は B です  
**Nghĩa**: A là B

```
わたし は がくせい です。
Tôi là học sinh.

これ は ほん です。
Đây là sách.
```

---

## 2. Câu Phủ Định với じゃありません
**Cấu trúc**: A は B じゃありません / ではありません  
**Nghĩa**: A không phải B

```
わたし は せんせい じゃありません。
Tôi không phải giáo viên.
```

---

## 3. Câu Hỏi với ですか
**Cấu trúc**: A は B ですか  
**Nghĩa**: A có phải B không?

```
あなた は にほんじん ですか。
Bạn là người Nhật phải không?
```

---

## 4. Quá Khứ với でした / じゃありませんでした
**Cấu trúc**: A は B でした / じゃありませんでした

```
きのう は にちようび でした。
Hôm qua là chủ nhật.

きのう は やすみ じゃありませんでした。
Hôm qua không phải ngày nghỉ.
```

---

## 5. の — Sở Hữu / Bổ Nghĩa
**Cấu trúc**: A の B  
**Nghĩa**: B của A

```
わたし の ほん = Sách của tôi
にほん の くるま = Xe của Nhật
```

---

## 6. も — Cũng
**Nghĩa**: Cũng

```
わたし も がくせい です。
Tôi cũng là học sinh.
```

---

## 7. これ / それ / あれ — Chỉ Thị
| Từ | Nghĩa | Khoảng cách |
|----|--------|------------|
| これ | Cái này | Gần người nói |
| それ | Cái đó | Gần người nghe |
| あれ | Cái kia | Xa cả hai |
| どれ | Cái nào | Câu hỏi |

```
これ は なん ですか。— Cái này là gì?
それ は とけい です。— Cái đó là đồng hồ.
```

---

## 8. ここ / そこ / あそこ — Nơi Chốn
| Từ | Nghĩa |
|----|--------|
| ここ | Ở đây |
| そこ | Ở đó |
| あそこ | Ở kia |
| どこ | Ở đâu |

```
トイレ は どこ ですか。— Nhà vệ sinh ở đâu?
あそこ です。— Ở kia.
```

---

## 9. Trợ Từ は / が
- **は**: Đánh dấu CHỦ ĐỀ (topic)
- **が**: Đánh dấu CHỦ NGỮ (subject)

```
わたし は たなか です。(Tôi là Tanaka — giới thiệu bản thân)
だれ が きましたか。(Ai đã đến? — hỏi chủ ngữ)
```

---

```

```

## 16. Động Từ ～ます (Thể Lịch Sự)

| Dạng | Ý nghĩa |
|------|---------|
| ～ます | Hiện tại/tương lai khẳng định |
| ～ません | Hiện tại/tương lai phủ định |
| ～ました | Quá khứ khẳng định |
| ～ませんでした | Quá khứ phủ định |

```
たべます → たべません → たべました → たべませんでした
```

## 21. あります / います — Có / Tồn Tại

- **あります**: Cho đồ vật, cây cối (không di chuyển)

```

にわ に ねこ が います。— Trong sân có mèo.
```

## 31. Te-form (て形) — Nền Tảng Quan Trọng

### Cách chia て-form:

**Nhóm I (五段動詞):**
| Đuôi | → Te form | Ví dụ |
|-------|-----------|-------|
| う、つ、る | → って | かう → かって |
| む、ぶ、ぬ | → んで | のむ → のんで |
| く | → いて | かく → かいて |
| ぐ | → いで | およぐ → およいで |
| す | → して | はなす → はなして |

**Ngoại lệ**: いく → いって

**Nhóm II (一段動詞):** Bỏ る, thêm て
- たべる → たべて
- みる → みて

**Nhóm III (Bất quy tắc):**
- する → して
- くる → きて

## 54. ～ないform (Thể Phủ Định Giản Thể)

| Nhóm | Cách chia | Ví dụ |
|------|-----------|-------|
| I | u→a + ない | かく → かかない |
| II | bỏ る + ない | たべる → たべない |
| III | する → しない, くる → こない | |

**Ngoại lệ**: ある → ない

## 65-80. Các Mẫu Bổ Sung

### 65. ～とき / ～ば / ～たら / ～なら — Điều Kiện
```
あめ が ふったら、いきません。— Nếu mưa, tôi không đi.
```

### 66. ～て、～て — Nối Hành Động
```
あさ おきて、かお を あらって、ごはん を たべます。
Sáng dậy, rửa mặt, ăn cơm.
```

### 67. ～や ～など — Liệt Kê Không Đầy Đủ
```
りんご や バナナ など を かいました。
Mua táo, chuối, v.v.
```

### 68. ～は ～が — Mô Tả Đặc Điểm
```
ぞう は はな が ながい です。
Voi thì mũi dài.
```

### 69. ～のは ～です — Nhấn Mạnh
```
いちばん すきな の は さくら です。
Cái tôi thích nhất là hoa anh đào.
```

### 70. ～かどうか — Có Hay Không
```
あした あめ が ふるかどうか わかりません。
Không biết ngày mai có mưa hay không.
```

### 71. ～し — Liệt Kê Lý Do
```
やすいし、おいしいし、この レストラン が すき です。
Vì rẻ, vì ngon, tôi thích nhà hàng này.
```

### 72. ～のに — Mặc Dù
```
べんきょう した のに、テスト に おちました。
Mặc dù đã học, nhưng trượt bài test.
```

### 73. ～ために — Để / Vì
```
にほん に いく ために、おかね を ためて います。
Để đi Nhật, tôi đang tiết kiệm tiền.
```

### 74. ～ようにする — Cố Gắng Làm
```
まいにち うんどう する ように して います。
Tôi cố gắng tập thể dục mỗi ngày.
```

### 75. ～ようになる — Trở Nên Có Thể
```
にほんご が はなせる ように なりました。
Tôi đã có thể nói tiếng Nhật.
```

### 76. ～すぎる — Quá
```
たべすぎました。— Ăn quá nhiều.
このふく は おおきすぎます。— Quần áo này quá lớn.
```

### 77. ～やすい / ～にくい — Dễ / Khó
```
この ほん は よみやすい です。— Sách này dễ đọc.
この かんじ は おぼえにくい です。— Kanji này khó nhớ.
```

### 78. ～かた — Cách Làm
```
つかいかた を おしえて ください。— Hãy chỉ cách sử dụng.
```

### 79. Thụ Động Cơ Bản ～(ら)れる
```
あめ に ふられました。— Bị mưa.
```

### 80. ～そうです (Dự Đoán) — Trông Có Vẻ
```
おいしそう です。— Trông có vẻ ngon.
あめ が ふりそう です。— Trông sắp mưa.
```
